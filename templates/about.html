RFI AI's Christmas Dream

This (rather hastily put together) website utilizes Neural Style Transfer and Google's Deep Dream algorithm to take an input image stylize it in a psychedelic way.

A neural network that has been trained to be able to classify different natural images, such as telling the difference between pictures of cats vs dogs, will have learned much about the textural and colour differences between images of cats and those of dogs.

Rather than asking the trained network to tell us which class it thinks and image is, we instead ask the network to tell us how to make the input image more like a cat or dog.

The result is the network pushes each of the coloured pixels in the direction that would make the whole image register stronger as that class. This is refered to as allowing the network to "dream".

This style of computer "dreamt" art is inspired by early works in science fiction such as Do Androids Dream of Electric Sheep?.
